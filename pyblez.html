PyBLEz Project

PyBLEz is a Python library for creating BLE peripherals using BlueZ and D-Bus. It simplifies the process of setting up BLE services and characteristics, handling read and write operations, and sending notifications to connected devices. This library was created to remove the complexity from BLE programming, making it accessible and efficient for developers.

Why Use PyBLEz?

  • Simplified BLE Development: Get started with BLE peripherals without diving into complex protocols.
  • Highly Customizable: Create and manage services and characteristics tailored to your needs.
  • Cross-Platform Support: Works on Linux systems that support BlueZ and Python.
  • Active Maintenance: Regular updates and community support ensure stability and reliability.

Key Features

  • Simple Service Creation: Add BLE services and characteristics easily.
  • Dynamic Interaction: Handle read, write, and notify operations with user-defined logic.
  • Efficient Advertising: Advertise your BLE peripheral with customizable parameters.

Installation

Install PyBLEz directly from PyPI:

pip install pyblez

Usage Example


#!/user/bin/env python3
from PyBLEz import create_ble_peripheral, enable_logs, disable_logs

# Enable debug logs
enable_logs()

def main():
    # create a BLE peripheral instance
    ble = create_ble_peripheral()
    ble.power_on_adapter()

    # Add a service
    service = ble.add_service("12345678-1234-5678-1234-56789abcdef0")

    # Add a characteristic to the service
    char = service.add_characteristic("12345678-1234-5678-1234-56789abcdef1", ["read", "write", "notify"], bytearray("Hello", "utf-8"))

    # Defind the read_value function
    def read_value(options):
        return char.value
    
    char.Read = read_value

    # Define the write_value function
    def write_value(value, options):
        data = bytes(value).decode("utf-8")
        print(f"Received: {data}")
        processed = data.upper()
        char.value = bytearray(processed, "utf-8")
        print(f"Processed: {processed}")

    char.Write = write_value

    # Register the GATT application
    ble.register_application()

    # Start advertising (10 seconds)
    ble.start_advertising("HelloBLE", [service.uuid], duration=10)

    # Run the main loop
    ble.run()

if __name__ == "__main__":
    main()
